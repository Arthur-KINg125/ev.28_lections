Linux commands

Ctrl+alt+t - commands for opening terminal 

~ - символ домашней дериктории 

/ - команда для ввывода дериктории 

. обозначает текушую дерикторию 

.. - обозначает пред. дерикторию  

1. Ls - команда для вывода всех дерикторий (папка) и файлов в текущей (рабочей) дериктории в терминале 

2.Cd (change directory) - команда для перехода из одной дериктории в другую

3.Mkdir - создание дериктории 

4.touch - создание файла

5.Rmdir - удаляет пустую дерикторию 

6.rm - удаляет файл 

7.rm -rf - удаляет всеееее

8.ls -a - показывает скрытые и обычные файлы и дериктории 
  ls -la - показывает подробную информацию 
  
  9. Pwd - показывает путь до текущей дериктории 
  
  10. cp -  копирует файл или дерикторию в указанное место 
  	cp < название > < куда >
 
 11.mv - перемещает или переименовывает файл или дериктории 
 	переименовывание: mv john.txt jamie.txt 
 	перемещение: mv john.txt directory/ 	пример: mv john.txt ../makers
 	
 12. cat - показывает содержимое файла 
 
 13. nano - открывает файл для редактирования в графическом интерфейсе Нано * если нет файла, то нано создаст новый
 
 14. echo - команда для быстрой записи текста в файл 
 	echo текст > файл
